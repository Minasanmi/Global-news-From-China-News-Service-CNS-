# 全球新闻速览

一个面向 Windows 桌面的新闻聚合应用，可实时查看国内和国际新闻，并支持搜索、筛选与自动刷新。

## 功能说明

- 聚合国内新闻与国际新闻
- 支持按分类切换查看
- 支持按标题、摘要、来源搜索
- 支持手动刷新
- 支持每 5 分钟自动刷新
- 支持点击新闻后打开原文链接

## 如何使用

1. 打开 `全球新闻速览.exe`
2. 左侧可以切换 `全部新闻`、`国内新闻`、`国际新闻`
3. 顶部搜索框可以输入关键词，快速筛选新闻
4. 点击 `立即刷新` 可以手动获取最新内容
5. 点击新闻卡片里的 `查看原文` 可以跳转到对应新闻页面

## 文件说明

- `全球新闻速览.exe`
  应用主程序，双击即可启动
- `resources\app\main.js`
  桌面应用主进程逻辑
- `resources\app\preload.js`
  渲染层与主进程通信桥接
- `resources\app\index.html`
  应用界面与前端交互逻辑
- `resources\app\package.json`
  应用基础配置

## 使用注意

- 请不要只单独移动 `全球新闻速览.exe`
- `exe` 需要和当前目录中的其他运行文件保持在一起，否则可能无法启动
- 首次打开时如果加载稍慢，通常是因为正在拉取新闻源
- 如果某一条新闻无法打开，通常是原始新闻链接已失效或新闻源暂时不可用
